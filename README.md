Global Sales Data Analysis (500K Records)

📌 Project Overview

This project analyzes a global sales dataset containing 500,000 records.
The objective of this project is to extract meaningful business insights using data analysis techniques.

The analysis is performed using Python, Pandas, NumPy, and Matplotlib.
---
📊 Dataset Information

The dataset contains the following columns:

- Region
- Country
- Item Type
- Sales Channel
- Order Priority
- Order Date
- Order ID
- Ship Date
- Units Sold
- Unit Price
- Unit Cost
- Total Revenue
- Total Cost
- Total Profit
---
🎯 Project Objectives

The main goal of this project is to solve real-world business problems using sales data.

Key objectives include:

1. Identify top performing countries based on total revenue
2. Determine the best selling products
3. Analyze online vs offline sales performance
4. Study regional sales performance
5. Understand the relationship between cost and profit
6. Analyze the impact of order priority on sales
---
🛠 Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook / Google Colab
---
📈 Data Analysis Performed
The following analyses were performed:
- Country-wise revenue analysis
- Product performance analysis
- Sales channel comparison
- Region-wise revenue and profit analysis
- Cost vs profit relationship analysis
---
📊 Data Visualization
Different types of visualizations were used in this project:
- Bar Charts
- Pie Charts
- Histograms
- Scatter Plots
These visualizations help in understanding trends and patterns in the sales data.
---
💡 Key Insights

Some important insights from the analysis include:

- Certain countries contribute the majority of total revenue.
- Some product categories have significantly higher sales.
- Online sales channels perform differently compared to offline channels.
- Sales performance varies across different regions.
